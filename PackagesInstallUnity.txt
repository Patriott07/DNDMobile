# How to add package 
- Download Package Dari unity
- Dan Lakukan Import 
- Tulis Nama package di file PackagesInstallUnity.txt

Apabila Setelah import terdapat folder baru dari package tersebut, ikuti step berikut :
-buka editor unity
-lakukan drag and drop folder package ke dalam 
Resource/Packages_Unity

- taruh di dalam file tersebut
